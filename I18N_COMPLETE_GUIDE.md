## 🔧 工作原理

### 1. 翻译字典

```javascript
translations = {
  zh: {
    "分析工作台": "分析工作台",
    "数据设置": "数据设置",
    ...
  },
  en: {
    "分析工作台": "Analysis Workbench",
    "数据设置": "Data Settings",
    ...
  }
}
```

### 2. 动态翻译

```javascript
// 监听语言切换
$(document).on("shiny:inputchanged", function(event) {
  if (event.name === "language_switcher") {
    translatePage(event.value);  // 立即翻译所有文本
  }
});
```

### 3. 文本匹配

- 自动遍历DOM树所有文本节点
- 精确匹配中文文本
- 实时替换为对应语言文本

### 3. 常见问题

**Q: 某些文本没有翻译？**
A: 需要将翻译键添加到 `modules/i18n_js.R` 的 `translations` 对象中。

**Q: 翻译不准确？**
A: 可以修改 `translations` 对象中的英文翻译值。

**Q: 如何添加新的翻译？**
A: 在 `translations` 的 `zh` 和 `en` 中分别添加中英文对照。

## 📝 添加新翻译的步骤

### 1. 找到需要翻译的文本

例如：侧边栏有 "我的新功能"

### 2. 在 `modules/i18n_js.R` 中添加

```javascript
translations = {
  zh: {
    "我的新功能": "我的新功能",  // 中文原文
    ...
  },
  en: {
    "我的新功能": "My New Feature",  // 英文翻译
    ...
  }
}
```

### 3. 重新加载应用

刷新浏览器或重启R session
