## 🎯 使用指南

### 新用户

**快速开始（3步）**:
1. 阅读 `START.md`
2. 运行应用（Windows: `launch_app.bat`）
3. 查看 `docs/user_guide/USER_MANUAL_COMPLETE.md`

**深入学习**:
- 第1-2章：项目介绍和快速开始
- 第3章：核心功能详解
- **第5章：富集分析最佳实践** ⭐⭐⭐⭐⭐

### 遇到问题

**解决流程**:
1. 查看 `USER_MANUAL_COMPLETE.md` 第6-7章（FAQ和故障排除）
2. 查阅 `BUG_SUMMARY_COMPLETE.md` 已知问题
3. 运行诊断工具 `diagnose_biofree()`
4. 查看 `KEGG_DEBUG_GUIDE.md`（KEGG相关问题）

### 开发人员

**日常工作**:
1. 熟读 `BUG_SUMMARY_COMPLETE.md`（所有已知Bug）
2. 使用诊断工具快速定位问题
3. 参考解决方案代码
4. 遵循最佳实践
