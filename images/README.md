# BioFastFree 图片资源文件夹

## 文件夹说明
此文件夹用于存放BioFastFree应用中使用的所有图片资源。

## 当前文件
- `logo.png` - 主要Logo文件 (已由用户添加)

## 图片要求

### Logo规格
- **格式**: PNG (推荐，支持透明背景)
- **尺寸**: 建议200x50像素
- **背景**: 透明或白色背景
- **文件大小**: <100KB (优化加载速度)

### 支持的格式
- PNG (推荐) - 支持透明背景，清晰度高
- JPG/JPEG - 适用于复杂图像
- SVG - 矢量图，可无限缩放
- ICO - 网站图标 (16x16, 32x32像素)

## 使用位置
1. **登录页面** - 大尺寸Logo显示 (最大高度80px)
2. **导航栏** - 小尺寸Logo显示 (高度30px)

## 容错机制
- 如果logo.png文件不存在，系统将自动显示DNA双螺旋图标 🧬
- 如果图片加载失败，也会回退到默认图标

## 添加更多图片
如需添加其他图片资源：
1. 将图片文件放入此文件夹
2. 在UI代码中引用: `tags$img(src="images/your-image.png")`

## 优化建议
- 使用PNG格式保持清晰度
- 控制文件大小，提高加载速度
- 提供深色和浅色两种版本以适应主题切换
- 确保logo在不同背景下都清晰可见
