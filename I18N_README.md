## 🔧 技术实现

### 方案选择

采用 **JavaScript 前端动态翻译** 方案：

**优点：**
- ✅ 无需修改现有模块代码
- ✅ 零学习成本
- ✅ 实时切换
- ✅ 性能优秀
- ✅ 易于维护

### 工作流程

```
用户切换语言
    ↓
JavaScript监听事件
    ↓
获取当前语言
    ↓
遍历DOM树所有文本节点
    ↓
匹配中文文本
    ↓
替换为目标语言
    ↓
完成（毫秒级）
```

### 代码示例

```javascript
// 监听语言切换
$(document).on("shiny:inputchanged", function(event) {
  if (event.name === "language_switcher") {
    translatePage(event.value);  // 立即翻译
  }
});
```

## 📝 使用说明

### 添加新翻译

如果发现某些文本未翻译，按以下步骤添加：

#### 1. 找到未翻译的文本

例如："我的新功能"

#### 2. 编辑 `modules/i18n_js.R`

```javascript
translations = {
  zh: {
    "我的新功能": "我的新功能",  // 添加这里
    ...
  },
  en: {
    "我的新功能": "My New Feature",  // 和这里
    ...
  }
}
```

#### 3. 重新加载应用

刷新浏览器或重启R session

### 批量添加翻译

如果要添加大量翻译，可以：

1. 先在 `modules/i18n.R` 中添加翻译键
2. 然后在 `modules/i18n_js.R` 中同步添加
3. 使用 `verify_i18n.R` 验证
