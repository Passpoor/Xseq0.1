## 🔧 技术实现

### 核心函数

```r
# 1. 静态翻译
text <- t_("login_title", "zh")
# 返回: "Biofree 生物信息学分析平台"

# 2. 响应式翻译
translator <- make_translator(current_language)
t <- translator()  # 绑定当前语言
text <- t("login_title")  # 自动使用当前语言
```

### 响应式更新

```r
# 初始化语言状态
current_language <- reactiveVal("zh")

# 监听语言切换
observeEvent(input$language_switcher, {
  current_language(input$language_switcher)
  showNotification("Language / 语言: ...")
})

# 自动更新UI
output$content <- renderUI({
  translator()()("my_key")  # 语言改变时自动重新渲染
})
```
